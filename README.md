# zcc-web-sdk-test

Static site for testing the Zoom ZCC Web SDK.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy to GitHub Pages

The site deploys automatically via GitHub Actions when you push to `main`. After enabling GitHub Pages in your repo settings (Settings → Pages → Source: GitHub Actions), it will be available at:

`https://<username>.github.io/zcc-web-sdk-test/`
